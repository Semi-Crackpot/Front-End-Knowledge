# 自适应两栏布局

### 要求
- 一个高度自适应的div，里面有两个子div，一个高度100px，实现另一个填满剩下的高度

### 布局
1.  绝对定位
  ```css
  .content {
    position: relative;
    height: 400px;
    width: 400px;
    border: 1px solid red;
  }
  .one {
    height: 100px;
    background-color: pink;
  }
  .sub {
    position: absolute;
    width: 400px;
    top: 100px;
    bottom: 0;
    background-color: blue;
  }
  ```
2.  calc布局
  ```css
  .content {
    height: 400px;
    width: 400px;
    border: 1px solid red;
  }
  .one {
    height: 100px;
    background-color: pink;
  }
  .sub {
    height: calc(100% - 100px);
    background-color: blue;
  }
  ```
3.  flex布局
  ```css
  .content {
    height: 400px;
    width: 400px;
    border: 1px solid red;
    display: flex;
    flex-direction: column;
  }
  .one {
    height: 100px;
    background-color: pink;
  }
  .sub {
    flex: 1;
    background-color: blue;
  }
  ```
