# 异步处理

### 通过async封装一个异步方法
    ```js
    async function test() {
        return new Promise((resolve, reject) => {
            setTimeout(function() {
                var name = 'wyz';
                resolve(name);
            }, 1000)
        })
    }
    ```

### 通过await获取异步方法中的数据
    ```js
    async function main() {
        var data = await test();
        console.log(data);
    }
    main();
    ```
